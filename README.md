# Static-vs-Responsive-Demo & Challenge — Style the Container

Your goal is to recreate the layout you see in the screenshot. You have the HTML and CSS starter files already. Do not change the HTML — all your work goes in the CSS file.

Here is what the finished result should look like:
- Four boxes sitting side by side, each taking up an equal amount of the screen width
- Box 1 is yellow
- Box 2 is green
- Box 3 is blue
- Box 4 is red
- Each box is tall enough to feel like a real section, not just a thin strip

**Requirements:**
- The four boxes must sit on the same row — no wrapping
- Each box must take up exactly 25% of the container width
- The height of each box must use a viewport unit
- Each box must have its correct background color

**Hints:**
- The parent container needs one property to line the boxes up side by side
- The width of each box has to be a percentage.
- For the height, think about the units you learned today — which one is based on the screen height?

**Stretch Goal:**
Once you have the layout working, add a `border` to each box. Does the layout break? What can you change to make sure the boxes are still next to eachother

# Challenge 4 — Add Text Styling

The text inside each box is hard to read on dark backgrounds. Fix it by doing the following:

- Change the text color inside every box to white
- Make the heading inside each box larger — try `2rem`
- Add some padding inside each box so the text is not pressed against the edge — try `1rem`

# Challenge 5 — Add a Title Above the Boxes

The `h2` that says "Style this container" is sitting right at the top of the page with no breathing room. Fix it by doing the following:

- Give the `h2` a font size of `3vw` — resize the window and watch what happens to it
- Add a bottom margin of `2vh` to push it away from the boxes
- Change the font color to whatever you like

# Challenge 6 — Make One Box Stand Out

Pick one box and make it feel different from the rest without changing its color. Try at least two of the following:

- Make its text bigger than the other boxes
- Give it a thick border in a contrasting color
- Make it taller than the other three using a different vh value
- Make its text bold or italic

There is no right answer here — just make it look intentional.
